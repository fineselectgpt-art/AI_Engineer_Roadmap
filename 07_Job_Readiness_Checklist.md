
# ✅ 07 — Job Readiness Checklist (AI Engineer)

This is your practical “ready to apply” system.

---

## 7.1 Portfolio Checklist (Non‑Negotiable)

- [ ] GitHub has **3+ projects**
- [ ] At least **2 are full apps** (API + demo)
- [ ] Each project has a clear README + run steps
- [ ] Each project includes evaluation (metrics + plots)
- [ ] At least one project is deployed

> ✅ If a recruiter clicks your repo, they should know what you built in 30 seconds.

---

## 7.2 Resume Checklist (AI Engineer)

Your resume should show:
- impact (metrics, latency, cost, accuracy)
- systems thinking (deployment, monitoring, CI)
- collaboration (PRs, docs, reviews)
- scope (end‑to‑end ownership)

### Bullet formula
**Action + Tech + Result + Context**
- Built a FastAPI inference service for sentiment classification, reducing manual triage time by 35% (demo + tests + Docker).

Templates in: `templates/Resume_Bullets.md`

---

## 7.3 Interview Prep (Technical)

### Topics you must be ready for
- ML fundamentals (bias/variance, leakage, metrics)
- deep learning basics (training loop, overfitting, regularization)
- data handling (pandas + SQL)
- deployment (API design, Docker, CI)
- LLM apps (RAG, embeddings, evaluation)

### Practice loop (weekly)
- 2 coding problems (Python)
- 1 ML concept explanation (write it)
- 1 project walkthrough (record yourself)
- 1 system design question (high level)

---

## 7.4 System Design (AI‑Flavor)

Be able to design:
- batch training pipeline
- online inference service
- monitoring + retraining triggers
- data quality checks

Template: `templates/ML_System_Design_Outline.md`

---

## 7.5 Behavioral Stories (STAR)

Prepare 6 stories:
- conflict / disagreement
- learning something hard fast
- shipping under deadline
- debugging a tricky issue
- improving a metric
- helping someone succeed

Template: `templates/STAR_Stories.md`

---

## 7.6 Job Search System (Simple but Effective)

### Weekly targets
- 10–20 applications (quality > quantity)
- 5 networking messages
- 1 portfolio improvement

### Track everything
Use a spreadsheet (template in `extras/Job_Search_Tracker.csv`).

---

## 7.7 Final “Ready” Gate

You’re ready when you can:
- demo each project smoothly
- explain metrics and tradeoffs
- answer “why this approach?” confidently
- describe limitations honestly
- show clean repos with documentation

---

## Checkpoint

- [ ] resume updated + tailored  
- [ ] LinkedIn updated + projects featured  
- [ ] 3 project walkthroughs practiced  
- [ ] mock interview completed  
- [ ] job tracker active and updated weekly
